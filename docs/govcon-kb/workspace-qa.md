# SAM.gov Workspace, Q&A, and documents

- Many DoD and civilian solicitations use **SAM.gov Workspace** (or agency-specific portals) for Q&A and submission. Submit questions only through the channel the solicitation specifies.
- **Amendments** can change due dates, requirements, or attachments—always work from the latest amendment set.
- If attachments fail to download (e.g. external host or auth wall), use the contracting officer’s stated fallback or paste key solicitation text for manual review.
