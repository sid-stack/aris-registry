# Advanced Usage

This guide covers advanced SDK features, customization, and integration tips.

## Custom Headers

```python
client = Aris(api_key="sk-...", headers={"X-Environment": "staging"})
```

## Streaming, async, and performance

- Use `AsyncAris` for high-throughput applications
- Prefer streaming if the node supports it to reduce latency
